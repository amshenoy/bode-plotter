# Bode Plotter

Asymptote and Real Plotter - JS Version

Simple Plotter - Python Version


Plotting by solving for quadratic zeros and poles.


### Example

Bode Plot 

```math
\frac{(s^2 + 2s + 3)(2s + 11)}{(s^2 + 3s + 2)(s^2 + 2s + 1)}
```

> ((s^2+2s+3)*(2s+11))/((s^2+3s+2)*(s^2+2s+1))

Numerator
```
1, 2, 3
0, 2, 11
```

Denominator
```
1, 3, 2
1, 2, 1
```
